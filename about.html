A B O U T

Our project is a Cyclone and thunderstorm intensity estimation model. Our model will predict the windspeed of the cyclones which further use to predict the destruction from that storms. Our project is based on an AI model called CNN (a type of deep learning model for processing data that has a grid pattern, such as images). It uses the INSAT3D imagery can be automated through the development of a software tool that integrates satellite imagery processing algorithms with a graphical user interface (GUI).

This software can allow users to quickly and easily access INSAT3D imagery, process the data to extract relevant information, and estimate the intensity of a cyclone based on a set of pre-determined algorithms.
